**Viterbi ICal service** Yue Wu This is a web service providing ICal file for Viterbi events. The service is available at [https://viterbi-ical.deno.dev](https://viterbi-ical.deno.dev). Usage ======================= The service provides the following endpoint: ```plain https://viterbi-ical.deno.dev/viterbi-calendar/{id: int} ``` The `id` is the id of the calendar. For example, for the CS Department, the id is `7`, meaning that the URL is `https://viterbi-ical.deno.dev/viterbi-calendar/7`.
